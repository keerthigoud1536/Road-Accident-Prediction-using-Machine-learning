# Road-Accident-Prediction-using-Machine-learning
This project focuses on analysing a real-world Accident Dataset to identify key factors that contribute to road accidents and to build machine learning models for predicting accident severity/outcomes.

**STEP 1:PERFORM COMPLETE DATA CLEANING**
- Handle Missing Values
- Remove Duplicates
- standardize categorical fields
- Fix Incorrect Dates ,Data types,Outliers
  
**STEP 2:EXPLORATORY DATA ANALYSIS**
- Accidents by Year,Month,State,City
- Severity Distribution
- Weather & Road Condition Impacts
- Time-based accident frequency
- Vehicle type analysis
- Drunk-driving involvement patterns
- Correlation heatmaps
  
**STEP 3:BUILD PREDICTIVEE MODELS**
- Acccident Severity Classification
- Likelihood of Alcohol Involvement
  
**STEP 4:CREATIVE VISUALIZATIONS**
- Heat Maps
- Filters
- Accident Trend lines
- Severity-level Comparisions
